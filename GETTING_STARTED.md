## ⚙️ API Key 配置说明

**重要提示**：API Key 不是必须的！CareerAssistant 的基础功能（个人评估、企业评估）可以在没有 API Key 的情况下正常使用。

### 什么时候需要 API Key？
- 使用 **智能推荐** 功能时
- 使用 **AI 洞察** 功能时
- 使用 **AI 问答** 功能时

### 如何配置 API Key？

1. 打开应用，进入 **系统设置**
2. 选择 **API Key 配置**
3. 选择 AI 提供商（DeepSeek 或 OpenAI）
4. 输入你的 API Key
5. 点击保存

**获取 API Key**：
- DeepSeek：访问 [platform.deepseek.com](https://platform.deepseek.com/)
- OpenAI：访问 [platform.openai.com](https://platform.openai.com/)

## 📊 核心功能详解

### 个人评估模块

#### MBTI 性格测试
- **题目数量**：28 题
- **测试时间**：约 5 分钟
- **结果类型**：16 种人格类型（如 INTJ、ENFP 等）
- **应用场景**：
  - 了解自己的沟通风格
  - 识别适合的工作环境
  - 发现职业倾向

#### 五大人格分析
- **维度**：开放性、尽责性、外向性、宜人性、神经质
- **测试时间**：约 8 分钟
- **结果应用**：
  - 评估岗位匹配度
  - 了解团队协作风格
  - 职业发展规划

#### 霍兰德职业兴趣
- **模型**：RIASEC 六边形模型
- **测试时间**：约 10 分钟
- **结果应用**：
  - 发现职业兴趣类型
  - 匹配相关行业和岗位
  - 职业满意度预测

### 企业评估模块

#### 公司信息管理
- **手动添加**：逐字段填写公司信息
- **批量导入**：支持 JSON/CSV 格式
- **字段说明**：
  - 公司名称、行业、规模
  - 融资阶段、地理位置
  - 稳定性评分、晋升清晰度
  - 标签、描述

#### 公司详情查看
- 公司基本信息
- 稳定性评分（0-100）
- 晋升路径分析
- 标签分类（稳定公司/新兴公司/分部公司）

### 智能推荐模块

#### 匹配算法
- **专业匹配度**：你的专业与岗位要求的契合度
- **性格匹配度**：性格类型与公司文化的适配度
- **兴趣匹配度**：职业兴趣与工作内容的契合度
- **年龄适配度**：年龄与岗位要求的匹配程度
- **地域适配度**：城市发展潜力与个人规划的契合度

#### 推荐结果
- **综合匹配分**：0-100 分的整体匹配度
- **匹配度卡片**：每个推荐公司的详细匹配分析
- **职业路径图**：可视化展示职业发展路径
- **行动建议**：具体的求职策略和技能提升建议

## 🔧 常见问题

### Q1：应用打开后是空白的？
**A**：这是正常现象，首次打开会显示新手引导。如果引导没有出现，请刷新页面或重启应用。

### Q2：没有 API Key 能用吗？
**A**：可以！基础功能（个人评估、企业评估）不需要 API Key。只有智能推荐和 AI 洞察功能需要。

### Q3：数据保存在哪里？
**A**：所有数据都保存在本地 SQLite 数据库中，位置：
```
Windows: %APPDATA%\career-assistant.db
Mac: ~/Library/Application Support/career-assistant.db
Linux: ~/.config/career-assistant.db
```

### Q4：可以导出数据吗？
**A**：可以。进入 **系统设置** → **数据备份** → **导出数据**，可以导出为 JSON 格式。

### Q5：如何添加示例公司数据？
**A**：
1. 进入 **系统设置** → **数据源管理**
2. 点击 **导入数据**
3. 选择 JSON 或 CSV 文件
4. 按照模板格式填写公司信息

### Q6：评估结果可以保存吗？
**A**：可以。完成评估后，结果会自动保存到本地数据库。你可以在 **个人评估** 页面查看历史记录。

### Q7：支持离线使用吗？
**A**：大部分功能支持离线使用，但智能推荐和 AI 洞察功能需要联网调用 API。

## 🎓 进阶使用

### 自定义评估问卷
如果你有自定义的评估问卷，可以：
1. 进入 `public/prompts/` 目录
2. 参考现有模板创建新的 JSON 文件
3. 在代码中加载自定义模板

### 批量导入公司数据
1. 准备 JSON 或 CSV 文件
2. 进入 **系统设置** → **数据源管理**
3. 点击 **导入** 选择文件
4. 系统会自动解析并导入

**JSON 格式示例**：
```json
[
  {
    "name": "字节跳动",
    "industry": "互联网",
    "scale": "large",
    "fundingStage": "已上市",
    "location": {
      "city": "北京",
      "district": "海淀区"
    },
    "stabilityScore": 85,
    "promotionClarity": 78,
    "tags": ["大厂", "稳定", "晋升快"],
    "description": "全球领先的互联网公司"
  }
]
```

### 配置自定义 API
如果你有自己的 AI API 服务：
1. 进入 **系统设置** → **爬虫配置**
2. 点击 **添加数据源**
3. 配置 API 地址、认证信息、请求格式
4. 保存后在推荐功能中使用
